# AI-Driven Threat Detection for Cloud Infrastructure

An automated security monitoring system that analyzes AWS cloud activity logs and detects suspicious behavior using machine learning.

---

## Overview

Cloud environments generate massive volumes of logs every minute, making manual monitoring impractical. This system continuously ingests cloud logs, identifies anomalous patterns using ML, and alerts administrators with AI-generated explanations of detected threats.

---

## Features

- Collects and processes logs from AWS CloudTrail
- Detects anomalies using machine learning (e.g., unusual login locations, abnormal API calls)
- Generates human-readable threat explanations via OpenAI API
- Triggers alerts and automated responses for suspicious activity
- Real-time dashboard for logs, anomalies, and threat alerts
